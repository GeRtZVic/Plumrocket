# module-swiss-post-graph-ql

### Version `1.0.0`

## Installation

In composer.json :
* create new entry in `repositories` node:
  ```
  {
  "type": "vcs",
  "url": "git@github.com:GeRtZVic/Release_plumrocket.git"
  }
  ```

* add new entry in `require` node:

  ```"plumrocket/module-reviews: "1.0.0"```
